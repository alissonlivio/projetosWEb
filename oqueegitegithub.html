O QUE É GIT E GITHUB?

  • O que é GIT?

É um sistema ou seja, um software que controla as versões de arquivos existentes, e permite que várias pessoas possam fazer edições, sem causar qualquer problema aos mesmos. É um sitema que trabalha utilizando códigos abertos, o que o torna bem cobiçado entre os desenvolvedores, geralmente é usado para armazenamento de dados em repositórios, que são guardados em servidores ou pastas. O Git é uma ferramenta muito útil, pois o mesmo mantém os registros das mudanças que são feitas, e quem as faz e também pode recuperar os dados editados. Para utilizar esse software é preciso baixa-lo, ele funciona como uma linguagem de edição, através de códigos simples, o que faz do mesmo uma ótima ferramenta. Podendo ser aplicado na nossa vida e no ambiente de trabalho.

  • O que é GITHUB?

É um sistema que gerencia os projetos e versões de códigos, é como se fosse uma rede social criada para que as pessoas, possam divulgar os seus projetos e trabalhos. O Github não pode ser instalado como o Git, ele funciona através de um serviço e para utiliza-lo, é preciso criar uma conta para ter acesso aos seus recursos.

  • Conclusão:
    •

O GIT e o GITHUB são ferramentas que estão bem interligadas entre si, como por exemplo: pense que o git fará o controle das edições dos arquivos e trabalhos, e o github como no local que será guardado todoso os projetos.
